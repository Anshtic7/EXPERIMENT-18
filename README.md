**THEORY**

Data visualization is an essential part of data analysis that helps in understanding patterns, trends, and relationships within datasets. In this experiment, various statistical and graphical techniques are implemented using Python libraries such as Matplotlib and Seaborn to represent data visually in different forms.

Matplotlib is a widely used plotting library that provides flexibility to create basic plots such as line graphs, area plots, pie charts, and scatter plots. It allows customization of plots including labels, colors, legends, and gridlines, making the visual representation more informative. Seaborn, built on top of Matplotlib, enhances visualization by providing more attractive and statistically informative plots such as heatmaps and boxplots.

Different types of plots serve different purposes. Line plots are used to show trends over time or continuous data. Area plots represent quantitative data cumulatively, highlighting magnitude. Pie charts and donut charts are used to show proportional data distribution. Boxplots are useful for statistical analysis, showing median, quartiles, and outliers in the dataset. Heatmaps represent data in matrix form using color intensity, making it easier to identify correlations or patterns. Bubble plots extend scatter plots by adding a third dimension using the size of bubbles.

These visualization techniques help in simplifying complex datasets into an understandable graphical format. They are widely used in fields such as data science, business analytics, and engineering to support decision-making and data interpretation.

**FUNCTION USED**

figure() – Creates a new figure for plotting graphs.

gcf() – Returns the current figure object.

gca() – Returns the current axes of the plot.

add_artist() – Adds a custom graphical element (artist) to the plot.

plot() – Draws a line graph for given data points. (if used implicitly)

fill_between() – Fills the area between two curves on a graph.

pie() – Creates a pie chart to represent proportional data.

Circle() – Creates a circular shape used for making donut charts.

scatter() – Plots data points in a scatter (bubble) plot.

scatterplot() – Creates an enhanced scatter plot using Seaborn.

boxplot() – Displays distribution of data using quartiles and outliers.

heatmap() – Visualizes data in matrix form using color intensity.

corr() – Computes correlation between variables in a dataset.

DataFrame() – Creates a structured data table using pandas.

head() – Displays the first few rows of a dataset.

legend() – Displays labels describing plotted data.

title() – Sets the title of the graph.

xlabel() – Labels the x-axis of the graph.

ylabel() – Labels the y-axis of the graph.

show() – Displays the final plot output.

set_style() – Sets the overall theme/style of Seaborn plots.

randint() – Generates random integer values.

seed() – Sets the seed for random number generation.
